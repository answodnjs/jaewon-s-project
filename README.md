# jaewon-s-project
휠체어를 타는 교통약자를 위한 프로젝트
시버스명: 휠체어에서 살아남기

이동약자 페르소나: 

페르소나

 새로운 장소를 방문할 때마다 엘리베이터나 경사로 등 접근성 정보를 미리 확인하기 어려워 불편을 겪는다. 그는 휠체어 이용이 가능한 이동 경로와 편의시설 정보를 한 번에 확인하여 안전하고 편리하게 이동할 수 있는 서비스를 필요로 한다.

어떤 서비스 인지: 계단이나 턱을 오를수있는 특수한 바퀴를 사용하고 버스나 지하철같은 대중교통안에서 안전하고 편리하게 이동하기위해서 휠체어의 크기가 몸에 딱 들어가게 작아지게하면 몸이 딱 껴서 휠체어에서 떨어지는 일도 적어지고 크기도 작아져 버스안에서도 이동이 편리해진다
